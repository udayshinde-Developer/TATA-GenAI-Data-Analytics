# TATA-GenAI-Data-Analytics #
📊 GenAI-Powered Credit Risk Analytics – Tata Job Simulation
🔍 Overview

Completed a job simulation by Tata Consultancy Services (TCS) focused on solving real-world credit risk and collections challenges for a fictional fintech company, Geldium, using Generative AI tools like ChatGPT and Copilot alongside data analytics techniques.

🧠 Key Objectives

Perform AI-assisted EDA on 500 customer records to identify credit delinquency risk factors
Design a binary classification model plan (Logistic Regression) to predict customer default
Generate business recommendations using the SMART framework for collections strategy
Build and present an Agentic AI-powered collections system to stakeholders


🛠️ Tools & Skills

ChatGPT, Copilot – Prompt engineering, insight generation, risk summarization
Python – ML pipeline design, feature engineering, model evaluation
Power BI / Excel – Data visualization and dashboarding
AI-Enhanced Reporting – SMART recommendations, business storytelling, ethical AI frameworks


📈 Outcome

Delivered a complete end-to-end analytics project — from raw data cleaning and EDA to predictive modeling, stakeholder reporting, and an AI-powered collections strategy — demonstrating how Generative AI boosts analyst efficiency in a real-world fintech environment.
